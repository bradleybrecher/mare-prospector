# 💎 MaRe Growth Catalyst: Autonomous Prospector
### **Pulse Miami Hackathon: The MaRe Scale-Up**

The **MaRe Growth Catalyst** is a purpose-built AI prospecting engine designed to identify, qualify, and engage luxury salons for the **MaRe Head Spa System**. Developed for the **Pulse Miami Hackathon**, this platform serves as an autonomous **Head of Biz Dev**, **Financial Analyst**, and **Real-Time Creative Director**, solving the "Who, Where, and How" of luxury B2B expansion.

## 🎯 The Mission
To scale MaRe’s footprint by targeting the $23B global scalp care market. Our engine identifies high-revenue salons ($1M+) that align with MaRe's aesthetic and possess the physical infrastructure to house the **MaRe Capsule**. It solves the "Personalized Scale Paradox" by automating discovery without sacrificing the $130/session luxury brand reputation.

## 🚀 Key Features

### 1. Autonomous Radar (Discovery)
* **Territory Scanning:** Uses **Firecrawl** to hunt for targets across any user-specified City/State.
* **Noise Filtering:** Intelligently bypasses directory sites like Yelp or Instagram to focus exclusively on direct salon domain data.
* **Contact Extraction:** Automatically scrapes the salon's website URL, public email, and phone number for immediate action.

### 2. The MaRe Audit (Qualification)
Every salon is passed through a **Gemini 2.5 Flash** reasoning engine programmed with a strict rubric based on the **M.A.R.E. Method**:
* **Prestige Index (1-100):** Grades salons based on "Neurosensory" language, ritual-based service menus, and luxury vocabulary.
* **Revenue Verification ($1M+):** Estimates annual revenue by analyzing stylist headcount, location count, and premium "backbar" pricing.
* **Infrastructure Viability:** Scans for "Infrastructure Clues" to verify room for the **MaRe Capsule** (minimum 108 sq. ft. required with plumbing).
* **AI Search Dominance:** Evaluates the salon's current structure for visibility in real-time AI search queries.

### 3. The Incentive Calculator (Financial Hook)
* **Retail Upsell Projections:** Automatically calculates the salon's missed revenue, pitching how the **MaRe Eye** camera increases retail conversion from the industry average of 3% to over 40%+.
* **ROI Timeline:** Generates a custom estimate on how many months it will take the salon to recover their investment based on their specific service volume.

### 4. Frictionless B2B Execution
* **1-Click Email Outreach:** Generates a hyper-personalized, 3-sentence "Human-in-the-Loop" script using authentic "Salon Lingo" (e.g., *retail velocity, backbar*). Clicking "✉️ Open in Email" instantly opens the user's default mail client with the target's email, subject line, and script perfectly pre-filled.
* **CRM-Ready CSV Export:** 1-click download of all audited leads (with contacts, scores, and scripts) for instant ingestion into HubSpot, Salesforce, or Apollo.
* **Content Engine Asset:** Generates a bespoke TikTok/Reel hook and concept for the salon, blending their aesthetic with MaRe's Zero-Gravity therapy to scale content output 50x.

### 5. Luxury Command Center UI
* The frontend is explicitly styled to match the **MaRe Brand Guidelines**, utilizing the official color palette (Key Teal, Brand Brown, Water-50) and typography (Playfair Display, Manrope).

## 🛠️ Tech Stack
* **Frontend:** Next.js / React
* **Engine:** Node.js
* **Scraping/Search:** Firecrawl SDK
* **Intelligence:** Google Gemini 2.5 Flash (Vertex AI)

## 📦 Installation & Usage

1.  **Clone & Install:**
    ```bash
    git clone [https://github.com/your-repo/mare-prospector.git](https://github.com/your-repo/mare-prospector.git)
    cd mare-prospector
    npm install
    ```

2.  **Configure Environment:**
    Ensure your `.env` contains `FIRECRAWL_API_KEY`, `GCP_PROJECT_ID`, and Google Cloud credentials.

3.  **Run the Prospector:**
    ```bash
    npm run dev
    ```
    Open `localhost:3000`. Enter a target City and State, click **Run Growth Catalyst**, and watch the engine hunt, audit, and generate turnkey B2B outreach in real-time.
