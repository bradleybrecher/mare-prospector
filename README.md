## 🚀 Key Features

### 1. Autonomous Radar (Discovery)
* **Territory Scanning:** Uses **Firecrawl** to hunt for targets across any user-specified City/State.
* **Noise Filtering:** Intelligently bypasses directory sites like Yelp or Instagram to focus exclusively on direct salon domain data.

### 2. The MaRe Audit (Qualification)
Every salon is passed through a **Gemini 2.5 Flash** reasoning engine programmed with a strict rubric based on the **M.A.R.E. [cite_start]Method**[cite: 39]:
* [cite_start]**Prestige Index (1-100):** Grades salons based on "Neurosensory" language, ritual-based service menus, and luxury vocabulary[cite: 23, 394].
* **Revenue Verification ($1M+):** Estimates annual revenue by analyzing stylist headcount, location count, and premium "backbar" pricing.
* [cite_start]**Infrastructure Viability:** Scans for "Infrastructure Clues" to verify room for the **MaRe Capsule** (minimum 108 sq. ft. required with plumbing)[cite: 91, 477, 479].
* [cite_start]**AI Search Dominance:** Evaluates the salon's current structure for visibility in real-time AI search queries[cite: 12].

### 3. Real-Time Creative Direction
* [cite_start]**Custom Content Assets:** Generates bespoke TikTok/Reel concepts that blend the salon’s unique aesthetic with the **MaRe Zero-Gravity** experience[cite: 100].
* [cite_start]**Bespoke Outreach:** Crafts high-conversion invitations using authentic **"Salon Lingo"** (e.g., *retail velocity, chair time, backbar*) to put meetings on the calendar[cite: 372, 387, 461].

## 🛠️ Tech Stack
* **Engine:** Node.js (v25.9.0)
* **Scraping/Search:** Firecrawl SDK
* **Intelligence:** Google Gemini 2.5 Flash (Vertex AI)
* **Data Pipeline:** Dynamic JSON generation for immediate frontend visualization.

## 📦 Installation & Usage

1.  **Clone & Install:**
    ```bash
    git clone https://github.com/your-repo/mare-prospector.git
    cd mare-prospector
    npm install
    ```

2.  **Configure Environment:**
    Ensure your `.env` contains `FIRECRAWL_API_KEY`, `GCP_PROJECT_ID`, and Google Cloud credentials.

3.  **Run the Prospector:**
    ```bash
    node index.js
    ```
    The interactive CLI will prompt you for a **City** and **State**. The engine then hunts, audits, and generates a `leads.json` file ready for the MaRe founders to review.
